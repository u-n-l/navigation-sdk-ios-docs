# Weather

The UNL Navigation SDK for iOS provides weather capabilities through its `WeatherContext` class. With it, you can retrieve current conditions, hourly forecasts, and daily outlooks — including temperature, air quality, atmospheric pressure, wind speed, UV index, and more.

- [Base Entities](/docs/11-Weather/01-Base%20Entities.md) - Weather-related data is organized into distinct classes, each designed to encapsulate specific weather information.
- [Weather Service](/docs/11-Weather/02-Weather%20Service.md) - The WeatherContext class provides methods for retrieving current, hourly, and daily weather forecasts. Access the shared instance via WeatherContext.shared().
