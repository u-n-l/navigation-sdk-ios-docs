# Display map items

This section covers how to render and manage map entities in iOS: landmarks, markers, overlays, routes, route instructions, and paths.

- [Display Landmarks](../docs/04-Maps/05-Display%20Map%20Items/01-Display%20Landmarks.md) - Learn how to filter, add, and highlight landmarks on the map.
- [Display Markers](../docs/04-Maps/05-Display%20Map%20Items/02-Display%20Markers.md) - Learn how to add and style point, polyline, and polygon markers, including labeling and clustering behavior.
- [Display Overlays](../docs/04-Maps/05-Display%20Map%20Items/03-Display%20Overlays.md) - Overlays provide enhanced, layered information on top of your base map.
- [Display Routes](../docs/04-Maps/05-Display%20Map%20Items/04-Display%20Routes.md) - Learn how to display routes on the map, assign the main route, style route rendering, and manage route labels.
- [Display Route Instructions](../docs/04-Maps/05-Display%20Map%20Items/05-Display%20Route%20Instructions.md) - Learn how to focus the map on route instructions and show maneuver arrows during route visualization.
- [Display Paths](../docs/04-Maps/05-Display%20Map%20Items/06-Display%20Paths.md) - Learn how to render PathObject instances on the map using PathCollectionObject.
